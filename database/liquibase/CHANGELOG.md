## Version Numbering Guidelines

- **MAJOR version** (X.0.0): Incompatible schema changes, breaking changes
- **MINOR version** (1.X.0): New features, backward-compatible changes
- **PATCH version** (1.0.X): Bug fixes, minor updates

## Categories

- **Added**: New features, tables, columns, etc.
- **Changed**: Changes to existing functionality
- **Deprecated**: Features that will be removed in future versions
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security-related changes
